# Legacy Problem Banks

This directory contains deprecated problem banks from the v1 curriculum.

## Deprecated Files
- **ProblemBank_A.md** (10 problems) - Replaced by ProblemBank_Python.md and ProblemBank_DSA.md
- **ProblemBank_B.md** (8 problems) - Replaced by ProblemBank_OOP.md and ProblemBank_DataHandling.md

## Why Deprecated?
The v2 curriculum (fundamentals-first, 6-week program) introduced new, more comprehensive problem banks:
- **ProblemBank_Python.md**: 25 problems (Python fundamentals)
- **ProblemBank_OOP.md**: 18 problems (OOP, design patterns)
- **ProblemBank_DSA.md**: 47 problems (data structures & algorithms)
- **ProblemBank_DataHandling.md**: 15 problems (CSV/JSON, ETL, validation)
- **ProblemBank_Django.md**: 22 problems (Django, DRF)

**Total: 127 problems** (vs. 18 in old banks)

## Should I Use These?
**No.** Use the new problem banks listed above. These files are kept only for reference.

If you find a problem here that's not in the new banks, please report it so we can add it to the appropriate new bank.

## Migration Notes
All unique problems from ProblemBank_A and ProblemBank_B have been incorporated into the new problem banks with:
- Better organization by topic
- Clearer input/output examples
- Complexity guidance
- Difficulty progression (Core → Stretch → Challenge)
